# Random Color Generator

A simple, interactive, lightweight web application that generates a random RGB color at the click of a button. The app dynamically updates the page's heading color, displays the generated RGB code, and fills a preview container with the new color.

# Features

* **Instant Color Generation:** Generates a completely randomized color using RGB channel values (0-255) with each click.
* **Dynamic UI Updates:** Simultaneously updates:
    * The background color of a dedicated preview canvas (`#color`).
    * The text color of the main page heading (`<h1>`).
    * The precise RGB value string inside an `<h3>` tag.
* **Clean Styling:** Center-aligned layouts featuring a clean color box container and an interactive button.
